# Advent of Code 2024

All in Rust, with [cargo-aoc](https://github.com/gobanos/cargo-aoc).

Run unit tests with example inputs:

```
cargo test --lib dayXX
```

Run solution on full input:

```
cargo aoc input -y 2024 [-d {day}]
cargo aoc -y 2024 [-d {day}]
```
